# Sicherheitsrichtlinie

## Unterstützte Versionen

Aktuell wird die jeweils neueste veröffentlichte Version mit 
Sicherheitsupdates versorgt.

## Sicherheitslücken melden

Sicherheitslücken bitte **NICHT** als öffentliches GitHub-Issue melden, 
um andere Nutzer nicht zu gefährden.

Stattdessen bitte:
1. Eine **private Sicherheitsmeldung** über GitHub erstellen 
   (https://github.com/EugHel/ets-gpa-sync/security/advisories/new)
2. Oder eine E-Mail über das GitHub-Profil senden

Ich antworte in der Regel innerhalb von 7 Tagen.

## Verantwortungsvolle Offenlegung

Bitte gib mir angemessene Zeit (mindestens 30 Tage), bevor du Details 
einer Sicherheitslücke veröffentlichst. Nach einem Patch dürfen Details 
gerne öffentlich diskutiert werden.

Danke, dass du zur Sicherheit dieses Projekts beiträgst! 🛡️
